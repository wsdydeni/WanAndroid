# WanAndroid

![](https://img.shields.io/github/languages/top/wsdydeni/WanAndroid) ![](https://img.shields.io/github/license/wsdydeni/WanAndroid)

## 项目简介

Jetpack MVVM 的最不佳实现

主要使用 Kotlin 语言，结合 MVVM + Kotlin Coroutine + Jetpack 打造的一个项目

## 主要功能

* 首页、知识体系、个人模块
* 登录注册功能
* 夜间模式/跟随系统
* 热门搜索、搜索历史
* 阅读置顶、横向收藏
* 每日 Github 项目推荐
* 听歌休闲模块
